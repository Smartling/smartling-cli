## smartling-cli jobs files remove

Remove files from a translation job.

### Synopsis

Detach files from an existing translation job. Each --file is a glob pattern matched against the project's files.

```
smartling-cli jobs files remove <translationJobUid|translationJobName> [flags]
```

### Examples

```

# Remove all XML files from a job

  smartling-cli jobs files remove aabbccdd1122 --file "**/*.xml"

# Remove a specific file from a job by name

  smartling-cli jobs files remove "Website Q1 2026" --file "menu/old.json"
```
