## smartling-cli jobs files add

Add files to a translation job.

### Synopsis

Attach files to an existing translation job. Each --file is a glob pattern matched against the project's files.

```
smartling-cli jobs files add <translationJobUid|translationJobName> [flags]
```

### Examples

```

# Add all JSON files to a job

  smartling-cli jobs files add aabbccdd1122 --file "**/*.json"

# Add files for specific locales

  smartling-cli jobs files add "Website Q1 2026" --file "menu/*.xml" --target-locale fr-FR

```
