## smartling-cli jobs strings remove

Remove strings from a translation job.

### Synopsis

Detach strings (by hashcode) from an existing translation job, identified by UID or name.

```
smartling-cli jobs strings remove <translationJobUid|translationJobName> [flags]
```

### Examples

```

# Remove two strings from a job

  smartling-cli jobs strings remove aabbccdd1122 --hashcode h1 --hashcode h2

# Remove a string from specific locales only

  smartling-cli jobs strings remove "Website Q1 2026" --hashcode h1 --locale fr-FR

```

### Options

